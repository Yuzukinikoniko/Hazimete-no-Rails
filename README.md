# Hazimete-no-Rails
Ruby on Railsを勉強する機会があり作ったものです。
簡単な勤怠管理を作成しました。
## 機能紹介
ログインすると、出勤になってログアウトすると退勤になるという簡単な作りです。
## 管理者画面
ホームでは一般利用の人がいつ出勤、退勤したのかを見ることができます。
